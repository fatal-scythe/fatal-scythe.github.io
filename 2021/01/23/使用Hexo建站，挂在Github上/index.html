使用Hexo建站，挂在Github上

2021-01-23 分类 杂七杂八 作者 Jae Fu 1129 字
本文最后编辑于 前，其中的内容可能需要更新。

一穷二白的开山之作

1、参考B站up主”CodeSheep”的视频，成功建站

       一个牛逼的同学给我推荐了CodeSheep的B站教学视频
搞了一晚上之后成功了，本来挂在github上，因为奇慢的速度让人心烦，就捣鼓了下放在gitee上了。
首先感觉速度杠杠的~
       然后就累了睡去…

2、网站迁移到github上

       第二天，在我绑定域名的时候发现，gitee的自定义域名功能和自动部署功能居然要收钱？？唉，算了，好像github白天挺快的，那就还是换回去吧~
       沿用从前的域名：iloveasuna.cn

3、SSH免密自动登录、theme选择

       就一直百度吧，遇到各种bug，终于实现了较为良好的开发流程，配置SSH免密部署，在 hexo d 的时候不用频繁输入密码了。
       然后在hexo的官方主题里面选了一款不错的，接下来的就是优化界面了~

4、基本开发流程

       使用hexo开发基本就用到这四个语句：

hexo new “blogName”   # 新建一篇博客
 hexo clean            # 清除什么东西不知道
 hexo g                # generate，把博客的md文件按照主题生成html+css+js文件
 hexo d                # 一件部署至github pages!

       首先 hexo new "blogName" 新建博客，这步会创建blogName.md文件，接下来编辑这个文件。（尝试着写了这一篇，感觉还行）
       然后再把上面剩下的3句挨个运行遍，完事~

5、这并不是教程

       下一步是逐渐移植我原来的网站的内容，同时推出更多的文章。欢迎关注呀~
